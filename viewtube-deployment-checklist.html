Phase 3: GoDaddy DNS Configuration
⚠️ CRITICAL: You must add TWO separate DNS records!

Add Domain to Vercel

Add your custom domain in Vercel first

📝 Steps:
  1. Vercel → Project Settings → Domains
  2. Click "Add" and enter: viewtube.live

Configure GoDaddy DNS Records

Log into GoDaddy and add these TWO records:

Record 1: A Record

Type:A
Name:@
Value:76.76.21.21
TTL:600

Record 2: CNAME

Type:CNAME
Name:www
Value:cname.vercel-dns.com
TTL:600
⚠️ Common Mistakes:
  • Don't select "A" for both records
  • Don't put "to:" before the value
  • Don't put IP in CNAME

Verify DNS & Enable HTTPS

Wait for DNS propagation and verify in Vercel

⏳ Wait Time: 5-30 minutes
📝 Steps:
  1. Wait 30 minutes
  2. Vercel → Project Settings → Domains → Verify
  3. HTTPS is automatic

Phase 6: YouTube Auth Integration

Enable YouTube Data API

Set up Google Cloud Console

📝 Steps:
  1. Go to console.cloud.google.com
  2. Select your project
  3. APIs & Services → Library
  4. Enable "YouTube Data API v3"

Configure OAuth Consent

Set up OAuth for YouTube authentication

📝 Steps:
  1. APIs & Services → OAuth consent screen
  2. User Type: "External"
  3. Add scopes: youtube.readonly
  4. Add test users

Create OAuth Credentials

Generate OAuth 2.0 Client ID

📝 Steps:
  1. APIs & Services → Credentials
  2. Create OAuth 2.0 Client ID
  3. Authorized origins: https://viewtube.live
  4. Redirect URI: https://viewtube.live/auth/callback
