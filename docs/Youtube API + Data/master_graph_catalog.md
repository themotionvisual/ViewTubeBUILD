# 📊 VIEWTUBE GOOGLE API MASTER GRAPH CATALOG
*The Ultimate Suite / Creator Command Center*

This catalog details the 7 definitive Google API charts that constitute the neo-brutalist ViewTube / UsTube dashboard. These are heavily stylized, API-driven visualizations built directly with `react-google-charts` and integrated tightly against the `WorkspaceBrain`.

---

## 🏛️ THE ULTIMATE SUITE

### 1. Top Performers Trio
*   **Google Chart Type:** Custom UI / High-Density Statistic Cards (400px height).
*   **Data Driven By:** 30-day cumulative algorithm performance.
*   **Visual Style:** Massive neo-brutalist border-boxes with centered statistics and satellite trend indicators.
*   **Strategic Purpose:** A massive revenue & reach overview. Acts as the heartbeat of the application, surfacing the core KPIs immediately on load.

### 2. Video Value Matrix
*   **Google Chart Type:** `ScatterChart` / `BubbleChart`.
*   **Data Driven By:** CTR % (*X-axis*) vs. Retention/AVP % (*Y-axis*). Bubble size scaled by Total Views.
*   **Visual Style:** Institutional quadrant overlays using reference lines (Crosshairs) to divide the chart into 4 distinct strategy zones.
*   **Strategic Purpose:** Quadrant-mapped performance classification:
    *   **GOLD MINE:** High CTR, High Retention.
    *   **HIDDEN GEM:** Low CTR, High Retention.
    *   **CLICK BAIT:** High CTR, Low Retention.
    *   **GRAVEYARD:** Low CTR, Low Retention.

### 3. Discovery Radar
*   **Google Chart Type:** `SteppedAreaChart` or `RadarChart` representation.
*   **Data Driven By:** Traffic origin flow (Browse vs. Search vs. External/Suggested).
*   **Visual Style:** Smooth, high-contrast overlapping polygons.
*   **Strategic Purpose:** Visualizes the traffic origin flow signature. Reveals the lifecycle phase of content—instantly identifying whether the channel is currently algorithmically "blessed" (heavy Browse) or reliant on search utility.

### 4. Algorithm Trigger
*   **Google Chart Type:** `ScatterChart`.
*   **Data Driven By:** Impressions vs. CTR.
*   **Visual Style:** Glowing data points mapping viral outliers.
*   **Strategic Purpose:** Maps "Viral Momentum." Isolates exact performance tipping points where high CTR forces massive impression delivery from the algorithm. 

### 5. Device Immersion
*   **Google Chart Type:** `PieChart` (with `is3D: true`).
*   **Data Driven By:** Viewing environment (Mobile vs. TV vs. Desktop).
*   **Visual Style:** Distinctly stylized 3D pie distribution to fit the neo-brutalist aesthetic.
*   **Strategic Purpose:** 3D Consumption breakdown. Dictates future formatting decisions (e.g., if TV dominates, text must be massive and pacing cinematic; if Mobile dominates, cuts must be extremely fast).

### 6. Narrative DNA
*   **Google Chart Type:** `WordTree`.
*   **Data Driven By:** Extracted semantics from top-performing video titles.
*   **Visual Style:** Expanding organic root structure.
*   **Strategic Purpose:** Semantic topic clustering. Finds the hidden keyword relationships that trigger clicks. If the tree heavily weights "Napoleon" -> "Marshals", the creator knows exactly what to title the next video.

### 7. Global Footprint
*   **Google Chart Type:** `GeoChart`.
*   **Data Driven By:** Views/Revenue mapped against geographic regions.
*   **Visual Style:** High-fidelity market heatmap using a 5-step vibrant transition (`Cyan` → `Lime` → `Yellow` → `Orange` → `Pink`).
*   **Strategic Purpose:** CPM-Optimized territory map. Dictates where translation and localization dollars should be spent by proving which geographic markets hold the highest financial density.
