Remotion Shapes.
https://www.remotion.dev/docs/shapes/

The following code shows a customizable react rectangle shape. You can adjust the width, currently set at 201, to be larger and smaller by changing the value. The corner radius and the height can be adjust as well. If these values are set to different numbers across a period of time, you can set the rectangle to expand, widen, shrink change its corner radius etc. Also it's settings for it's position could be adjusted in this same time period, creating a rectangle that, for instance, slides to the left while expanding, or/ and changing color etc and then text could be applied top of it as well.

Many other properties, like transparency, stroke width etc can also be adjusted. An Ai llm or skills based generator with system instructions could be used to take these functions and this tool to generates assets for a YouTube video or channel with any text a creator might let. A tool generating these assets could be controlled 100% by prompt, 100% by user adjusting and controlling the settings for themselves, or any combination of the two. Many templates could be included in such a tool as well, and users could exchange and create their own templates too. This rectangle information is at https://www.remotion.dev/docs/shapes/rect but there are many other shapes in the documentation as well
