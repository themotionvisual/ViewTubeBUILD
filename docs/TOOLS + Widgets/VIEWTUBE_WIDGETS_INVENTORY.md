## 🔧 WIDGET DEVELOPMENT GUIDELINES

### Code Standards
- **Max Lines**: 500 lines per widget
- **Props Interface**: TypeScript strict typing
- **Styling**: Tailwind CSS with Neo-Brutalist theme
- **Testing**: Unit tests for interactive widgets
- **Documentation**: JSDoc comments required

### Performance Requirements
- **Render Time**: < 50ms for micro widgets
- **Re-render Optimization**: Use React.memo for expensive widgets
- **Lazy Loading**: Load complex widgets on demand
- **Bundle Size**: Keep individual widgets < 50KB gzipped

### Accessibility Standards
- **Keyboard Navigation**: Full keyboard support
- **Screen Reader**: ARIA labels for all interactive elements
- **Color Contrast**: WCAG AA compliance minimum
- **Focus Management**: Clear focus indicators

## 🎨 NEO-BRUTALIST WIDGET STYLING

### Standard Widget Styles
```css
/* Widget Container */
.widget-container {
  @apply border-[4px] border-black rounded-3xl;
  @apply shadow-[6px_6px_0px_0px_black];
  @apply bg-white dark:bg-gray-900;
  @apply p-4;
}

/* Widget Header */
.widget-header {
  @apply font-[1000] uppercase tracking-tighter;
  @apply text-lg mb-3;
  @apply border-b-[3px] border-black pb-2;
}

/* Widget Content */
.widget-content {
  @apply text-sm leading-relaxed;
}

/* Interactive Elements */
.widget-button {
  @apply border-[3px] border-black rounded-xl;
  @apply shadow-[4px_4px_0px_0px_black];
  @apply hover:shadow-[2px_2px_0px_0px_black];
  @apply active:translate-x-[2px] active:translate-y-[2px];
  @apply transition-all duration-75;
}
```
